# Project Context

This project revisits the January 2022 paper **Transitory Inflation and Projection of Future Inflation** by Matt Peron and Yosef Bonaparte.

The original research idea is useful because it converts a vague macro phrase, "transitory inflation," into a measurable time-series state variable.

The user wants:

- a clean Python re-implementation,
- a professional macro/trader research expansion,
- a local Streamlit app to eyeball before Git push,
- a Claude Code workflow that asks when ambiguous,
- enough structure for agentic work without forcing every visualization/design choice.

## Research attitude

Treat the paper as a reference, not as unquestionable truth. Replicate first, then improve.
